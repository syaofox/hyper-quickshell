yazi

```
在 CachyOS (Arch 系) 上，执行：
sudo pacman -S yazi ffmpeg 7zip jq poppler fd ripgrep fzf zoxide resvg imagemagick
Wayland 下还需安装剪贴板支持：
sudo pacman -S wl-clipboard
如需最新 Git 版（通过 AUR）：
paru -S yazi-git
或者官方 nightly 二进制：
paru -S yazi-nightly-bin

```
